== Description ==

No more ugly Facebook Fan Box, stop paying for Fan Box, Ultimate FaceBook Fan Box provided by PicaJob is 100% free and can be customized to fit your Wordpress design. Using an appealing design Ultimate FaceBook Fan Box turn your visitors into Fans. Improve your Brand name Use Ultimate FaceBook Fan Box.

* Custom width of the FanBox.
* Custom height of the FanBox.
* Custom FanBox background color.
* Custom FanBox border color.
* Custom FanBox font color.
* Custom FanBox font size.
* Custom Fan's image width.
* Custom Fan's link color.
* Custom distance between images.

To configure the plugin:

1. Go to Appearance > Widgets
2. Paste Ultimate FaceBook Fan Box on your Sidebar
3. The options will show up.


== Installation ==

1. Upload the plugin to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Appearance > Widgets
4. Drop the widget on your side and continue!

After installing Ultimate FaceBook Fan Box you need to get your Facebook fan page ID. 
To get that ID you need to inspect urls inside your Fan Page, the ID is the last digits of the url.

To configure the plugin:

1. Go to Appearance > Widgets
2. Paste Ultimate FaceBook Fan Box on your Sidebar
3. The options will show up.
